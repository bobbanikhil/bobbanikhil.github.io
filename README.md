## 🚀 LOCAL SETUP

### Prerequisites
```bash
Node.js ≥18.0 | npm ≥9.0
```

### Installation
```bash
# Clone repository
git clone https://github.com/bobbanikhil/bobbanikhil.github.io.git
cd bobbanikhil.github.io

# Install dependencies
npm install

# Start development server
npm run dev
# Runs at http://localhost:3000 with HMR

# Build for production
npm run build

# Preview production
npm run preview

# TypeScript validation
npm run lint
```
